# 🚀 超简单启动指南

## ⚠️ 重要提醒

### ❌ 不要使用conda！
```bash
# 错误的做法：
conda activate .glut  # ❌ 项目不使用conda！
```

### ✅ 正确启动方法

## 第1步：启动后端
**在主目录** `OpenAvatarChat-250916\` 双击：
```
04运行程序-lam-VL.bat    # 推荐：LAM 3D + 视觉理解
```

## 第2步：启动前端  
**进入WebUI目录** `OpenAvatarChat-250916\OpenAvatarChat-WebUI\` 双击：
```
start-dev.bat
```

## 第3步：访问
- 前端：http://localhost:5173
- 后端：https://127.0.0.1:8282

## 🎯 故障排除

### 如果看到编码错误：
- ✅ 已修复！新脚本使用英文避免编码问题

### 如果提示目录错误：
- 确保在 `OpenAvatarChat-WebUI\` 目录运行 `start-dev.bat`
- 不要在主目录 `OpenAvatarChat-250916\` 运行前端脚本

### 如果conda报错：
- 完全忽略！项目不需要conda环境
- 直接双击启动脚本即可

## ✅ 成功标志

**后端启动成功**：看到 "服务已启动" 或访问 https://127.0.0.1:8282 有页面

**前端启动成功**：看到 "Local: http://localhost:5173" 并能访问
